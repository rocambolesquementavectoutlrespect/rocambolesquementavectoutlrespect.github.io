Formes du trinôme
Forme développée
f(x) = ax² + bx + c

avec a ≠ 0, a, b, c ∈ ℝ

Forme canonique
f(x) = a(x − α)² + β

α = −b/(2a) est l'axe de symétrie
β = f(α) est le sommet

Forme factorisée
f(x) = a(x − x₁)(x − x₂)

Possible uniquement si Δ ≥ 0. x₁ et x₂ sont les racines.

Discriminant Δ
Calcul
Δ = b² − 4ac
1
Identifier a, b, c dans f(x) = ax² + bx + c
2
Calculer b² puis soustraire 4ac
3
Analyser le signe de Δ
Solutions selon Δ
CasRacines
Δ > 02 racines réelles distinctes
x₁,₂ = (−b ± √Δ) / 2a
Δ = 01 racine double
x₀ = −b / 2a
Δ < 0Aucune racine réelle
Δ > 0 : x₁ = (−b − √Δ)/2a  |  x₂ = (−b + √Δ)/2a
Signe du trinôme & concavité
Règle générale
a > 0 : parabole tournée vers le haut (concave vers le haut)
f(x) est du signe de a à l'extérieur des racines.
a < 0 : parabole tournée vers le bas (concave vers le bas)
f(x) est du signe de a entre les racines.
Résumé : f(x) est du signe de a sauf entre x₁ et x₂ (si Δ > 0).

Méthode — Inéquations du 2nd degré
1
Ramener l'inéquation à f(x) > 0 (ou < 0)
2
Calculer Δ, trouver x₁ et x₂ si Δ ≥ 0
3
Dresser le tableau de signes de f(x)
4
Lire la solution sur le tableau (intervalles)
Exemple : x² − 3x + 2 < 0
Δ = 9 − 8 = 1 > 0 → x₁ = 1, x₂ = 2
Solution : x ∈ ]1 ; 2[
Récapitulatif — Trinôme du second degré
  • f(x) = ax² + bx + c avec a ≠ 0
  • Δ = b² − 4ac détermine le nombre de racines réelles
  • Δ > 0 : x₁,₂ = (−b ± √Δ) / 2a
  • Forme canonique : f(x) = a(x − α)² + β, sommet S(α, β)
  • Signe de f : du signe de a sauf entre les racines (si Δ > 0)
  • Concavité : vers le haut si a > 0, vers le bas si a < 0
  • Inéquation : tableau de signes → lire les intervalles solutions

ASUKA LANGLEY
02
Définition — Nombre dérivé
Taux de variation
τ = [f(a + h) − f(a)] / h

Taux de variation de f entre a et a+h.

Dérivée en a
f'(a) = limh→0 [f(a+h) − f(a)] / h

f est dérivable en a si cette limite existe et est finie.

a 3.0
Équation de la tangente
Formule
y = f'(a)(x − a) + f(a)

Tangente à la courbe de f au point d'abscisse a.
Pente (coefficient directeur) : m = f'(a)

Interprétation géométrique
  • f'(a) > 0 → tangente montante
  • f'(a) < 0 → tangente descendante
  • f'(a) = 0 → tangente horizontale (extremum local possible)
Méthode — Trouver l'équation de tangente
1
Calculer f(a) : ordonnée du point de tangence
2
Calculer f'(x) puis f'(a) : pente de la tangente
3
Substituer dans y = f'(a)(x − a) + f(a)
Exemple : f(x) = x², a = 3
f(3) = 9  |  f'(x) = 2x → f'(3) = 6
Tangente : y = 6(x − 3) + 9 = 6x − 9
Application aux variations
f croissante sur I
f'(x) > 0 sur I

La dérivée est strictement positive sur l'intervalle.

Extremum local en a
f'(a) = 0

Et f' change de signe en a. Maximum si f' passe de + à −. Minimum si f' passe de − à +.

f décroissante sur I
f'(x) < 0 sur I

La dérivée est strictement négative sur l'intervalle.

Trouver une équation à partir d'une tangente
Méthode inverse

On connaît la tangente y = mx + p en x = a. Alors :

f(a) = ma + p  (ordonnée du point)
f'(a) = m  (pente de la tangente)

Ces deux conditions permettent de déterminer les constantes d'une famille de fonctions.

Exemple :
f(x) = ax² + b. La tangente en x = 2 est y = 4x − 3.
f(2) = 4(2) − 3 = 5 → 4a + b = 5
f'(x) = 2ax → f'(2) = 4a = 4 → a = 1
Donc b = 1. f(x) = x² + 1
Récapitulatif — Dérivée & Tangente
  • f'(a) = lim(h→0) [f(a+h) − f(a)] / h
  • Équation de tangente en a : y = f'(a)(x − a) + f(a)
  • f'(x) > 0 ⟹ f croissante  |  f'(x) < 0 ⟹ f décroissante
  • f'(a) = 0 avec changement de signe ⟹ extremum local en a
  • Pente de la tangente = f'(a) = coefficient directeur

Unique fonction dérivable sur ℝ telle que f' = f et f(0) = 1.

Exemples de calcul de dérivée
Exemple 1 — Produit
f(x) = x²·eˣ
u = x² → u' = 2x
v = eˣ → v' = eˣ
f'(x) = 2x·eˣ + x²·eˣ = eˣ(2x + x²)

f(x) = eˣ / x
u = eˣ → u' = eˣ
v = x → v' = 1

Exemple 4 — Racine composée

u = x² + 1 → u' = 2x
f'(x) = 2x / (2√(x²+1)) = x/√(x²+1)

KAWORU NAGISA

Suite arithmétique
Définition
uₙ₊₁ = uₙ + r

r = raison (constante). Chaque terme s'obtient en ajoutant r au précédent.

Terme général
uₙ = u₀ + n·r  ou  uₙ = u₁ + (n−1)·r
Somme de n+1 termes (de u₀ à uₙ)
S = (n+1) · (u₀ + uₙ) / 2

= (nb de termes) × (premier + dernier) / 2

Méthode de Gauss — Somme 1 + 2 + … + n
S = n(n+1)/2
Exemple : 1 + 2 + 3 + … + 100
S = 100 × 101 / 2 = 5050
Reconnaître une suite arithmétique
Calculer uₙ₊₁ − uₙ.
Si le résultat est constant, la suite est arithmétique de raison r = uₙ₊₁ − uₙ.
Graphe

Les points (n, uₙ) sont alignés. La représentation graphique est une droite.

Suite géométrique
Définition
uₙ₊₁ = uₙ × q

q = raison (constante, q ≠ 0). Chaque terme s'obtient en multipliant par q.

Terme général
uₙ = u₀ × qⁿ  ou  uₙ = u₁ × qⁿ⁻¹
Somme de n+1 termes (de u₀ à uₙ)
S = u₀ × (1 − qⁿ⁺¹) / (1 − q)  si q ≠ 1
Exemple — 6 + 12 + 24 + … + 6·2⁹
1
Identifier : u₀ = 6, q = 2, n+1 = 10 termes
2
Appliquer S = 6 × (1 − 2¹⁰) / (1 − 2)
3
S = 6 × (1 − 1024) / (−1) = 6 × 1023 = 6138
Reconnaître une suite géométrique
Calculer uₙ₊₁ / uₙ.
Si le résultat est constant, la suite est géométrique de raison q = uₙ₊₁/uₙ.
Suites ni arithmétiques ni géométriques
Principe

Certaines suites ne sont ni arithmétiques ni géométriques. Dans un exercice, une seconde suite auxiliaire sera donnée pour t'aider à exprimer le terme général.

Démarche type
1
On te donne une suite (uₙ) définie par récurrence (ex: uₙ₊₁ = 2uₙ + 3)
2
L'énoncé introduit une suite auxiliaire (ex: on pose vₙ = uₙ + 3)
3
Tu montres que (vₙ) est géométrique en calculant vₙ₊₁ / vₙ
4
Tu exprimes vₙ puis uₙ = vₙ − 3 pour retrouver le terme général
Exemple guidé — uₙ₊₁ = 2uₙ + 3, u₀ = 1
Énoncé : On pose vₙ = uₙ + 3

vₙ₊₁ = uₙ₊₁ + 3 = (2uₙ + 3) + 3 = 2uₙ + 6 = 2(uₙ + 3) = 2vₙ
→ (vₙ) est géométrique de raison q = 2
v₀ = u₀ + 3 = 1 + 3 = 4
vₙ = 4 × 2ⁿ

Donc uₙ = vₙ − 3 = 4 × 2ⁿ − 3

Fiche 05 · Dénombrement & Probabilités
Probabilités
Événements, arbres, calculs, indépendance
MISATO
05
Vocabulaire & Événements
Univers Ω

Ensemble de toutes les issues possibles d'une expérience aléatoire.

P(Ω) = 1  |  P(∅) = 0
Événement contraire Ā
P(Ā) = 1 − P(A)

A et Ā sont incompatibles : A ∩ Ā = ∅

Union & Intersection
P(A∪B) = P(A) + P(B) − P(A∩B)

Si A et B incompatibles : P(A∪B) = P(A) + P(B)

Probabilité conditionnelle & Indépendance
Probabilité conditionnelle
P(B|A) = P(A∩B) / P(A)

Probabilité de B sachant que A est réalisé. Nécessite P(A) > 0.

Reformulation :
P(A∩B) = P(A) × P(B|A) = P(B) × P(A|B)
Événements indépendants
P(A∩B) = P(A) × P(B)

A et B sont indépendants si et seulement si la réalisation de l'un ne modifie pas la probabilité de l'autre.

Équivalent : P(B|A) = P(B)
Si indépendants : P(A∩B) = P(A)·P(B)
Arbre de probabilités
Règles : multiplier sur les branches · additionner les branches menant au même événement
Règles de lecture
Multiplication : le long d'une branche
P(A∩B) = P(A) × P(B|A)
Addition : entre les branches aboutissant au même résultat
P(B) = P(A∩B) + P(Ā∩B)
Vérification : la somme de toutes les probabilités des feuilles = 1
Formule des probabilités totales
P(B) = P(A)·P(B|A) + P(Ā)·P(B|Ā)
Exemples de calcul
Exemple 1 — Tirages avec remplacement

Urne : 3 boules rouges, 2 bleues. Deux tirages avec remise.

P(rouge) = 3/5  |  P(bleue) = 2/5
P(rouge et rouge) = 3/5 × 3/5 = 9/25
(tirages indépendants car avec remise)
Exemple 2 — Tirages sans remplacement

Même urne, deux tirages sans remise.

P(2ème rouge | 1er rouge) = 2/4 = 1/2
P(rouge ∩ rouge) = 3/5 × 1/2 = 3/10
(non indépendants car sans remise)
Récapitulatif — Probabilités
  • P(Ā) = 1 − P(A)  |  P(Ω) = 1  |  P(∅) = 0
  • P(A∪B) = P(A) + P(B) − P(A∩B)
  • P(B|A) = P(A∩B) / P(A) — probabilité conditionnelle
  • A, B indépendants ⟺ P(A∩B) = P(A)·P(B)
  • Arbre : multiplier sur une branche, additionner pour un même événement
  • P(B) = P(A)·P(B|A) + P(Ā)·P(B|Ā) — probabilités totales
